# Credit Scoring Model using Gradient Boosting

Проект посвящен разработке модели кредитного скоринга для прогнозирования вероятности дефолта заемщика на основе исторических данных банка (150 000 записей).

## Цель проекта
Построить бинарный классификатор, определяющий клиентов, которые могут допустить просрочку выплаты кредита более чем на 90 дней в течение следующих двух лет (целевой признак: `SeriousDlqin2yrs`).

## Особенности датасета и предобработка (EDA)
В ходе разведочного анализа данных (EDA) были решены следующие ключевые проблемы:
* **Пропуски в данных:** Выявлена скрытая логика в признаке `DebtRatio` — у клиентов с пропущенным доходом (`MonthlyIncome`) в этой колонке содержалась абсолютная сумма долга. Пропуски доходов восстановлены путем группировки заемщиков по возрасту и количеству кредитных линий с последующим расчетом медианы внутри групп. Пропуски в количестве иждивенцев заполнены нулями.
* **Экстремальные выбросы:** Исключены аномальные значения ежемесячного дохода (ошибки ввода данных свыше $3 000 000). Датасет ограничен по 99.9-му процентилю доходов ($72 000).
* **Дисбаланс классов:** Доля дефолтных клиентов в выборке составляет ~6.6%. В связи с этим в качестве основной метрики эффективности выбран ROC-AUC, а разбиение на обучающую и тестовую выборки производилось со стратификацией.

## Моделирование и результаты
В рамках проекта были протестированы два подхода к классификации:
1. **Логистическая регрессия (Baseline):** Требовала предварительного масштабирования признаков (`StandardScaler`). Из-за сложной нелинейной структуры данных показала слабый результат ранжирования: **ROC-AUC ~69.7%**.
2. **Гистограммный градиентный бустинг (`HistGradientBoostingClassifier`):** Использован алгоритм на основе деревьев решений, устойчивый к разному масштабу признаков и нелинейным зависимостям. Попытки ручной генерации признаков (Feature Engineering) и тюнинг гиперпараметров не привели к существенному улучшению базовой конфигурации алгоритма.

### Финальные метрики модели:
* **ROC-AUC:** 86.45%
* **Accuracy:** 93.64%
